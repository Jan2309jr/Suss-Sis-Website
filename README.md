# Suss Sis Gourmet Bakery & Cafe

A full-stack web application for a bakery cafe, built with React, Vite, Express, and SQLite.

## Features

- Menu management
- Order system
- Admin panel
- User authentication
- Custom cake orders
- Gallery
- Contact form

## Tech Stack

- Frontend: React, TypeScript, Vite, Tailwind CSS
- Backend: Express.js, Node.js
- Database: SQLite with Drizzle ORM
- Deployment: Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This app is configured for Vercel deployment with serverless API routes.
